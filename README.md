# 小3ぼうけんドリル

小学3年生向けの、毎日10分で続けやすい学習サイトです。
静的サイトなので、サーバー不要で `index.html` を開くだけで動きます。
Vercel / Netlify / GitHub Pages にそのままアップロードできます。

## 入っている機能

- 今日のミッション
  - 算数・国語・理科・社会を自動でミックス
  - 5問 / 8問 / 10問 / 12問に変更可能
- 教科別クイズ
  - 算数の森
  - 漢字道場
  - 理科ラボ
  - 町たんけん
- 単元別練習
- ミニ授業ページ
- にがてボックス
  - 間違えた問題だけ自動保存
  - 正解すると苦手回数が減る
- おうちの人ページ
  - 正答率
  - 教科別集計
  - 苦手単元
  - 今週の学習計画
  - 学習データの書き出し
- レベル・XP・コイン・バッジ
- スマホ対応
- localStorage保存
  - サーバーにデータ送信しません

## ローカルで見る方法

一番簡単な方法：

1. フォルダを開く
2. `index.html` をダブルクリック

ブラウザの制限で動きが変な場合は、フォルダ内で簡易サーバーを起動します。

```bash
python -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

## Vercelで公開する方法

1. GitHubにこのフォルダをアップロード
2. Vercelで「New Project」
3. 該当リポジトリを選択
4. Framework Preset は `Other`
5. Build Command は空欄
6. Output Directory も空欄または `.`
7. Deploy

## Netlifyで公開する方法

1. Netlifyにログイン
2. 「Add new site」→「Deploy manually」
3. このフォルダをドラッグ＆ドロップ

## 問題を追加する方法

`app.js` の上のほうにある `QUESTIONS` に追加します。

```js
{
  "id": "q999",
  "subject": "math",
  "unit": "わり算",
  "level": 1,
  "question": "32 ÷ 4 = ？",
  "choices": ["6", "7", "8", "9"],
  "answer": "8",
  "explanation": "4×8=32なので、32÷4=8です。",
  "hint": "わり算は、かけ算にもどして考えよう。"
}
```

### subject の種類

```text
math      算数
japanese  国語
science   理科
social    社会
```

## 作り込み候補

次に作り込むなら、以下がおすすめです。

1. 問題数を500問以上に増やす
2. 漢字を教科書の進度別に分ける
3. 音声読み上げを入れる
4. 手書き漢字練習ページを作る
5. 親用にCSV出力を追加する
6. Firebaseなどで複数端末同期する
7. キャラ・マップ・ボス戦をもっとゲーム化する

## 注意

現在は家庭学習用のたたき台です。
学校や教科書によって進度が違うので、実際に使うときは単元・漢字・用語を調整してください。
